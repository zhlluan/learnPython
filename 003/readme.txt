题目1：一个整数，它加上100后是一个完全平方数，再加上168又是一个完全平方数，请问该数是多少？


x + 100 = y*y
x + 100 + 168 = z*z
=> z*z - y*y = 168 
=> (z+y)*(z-y)=168
把z+y 和 z-y分别看成变量a, b，那么a和b在负168到正168之间是没问题的。

x=(y*y+z*z)/2-184
y、z取正值和负值, x都是同一个结果
故y、z的取值范围可以定义为0到168之间。

执行结果：
-99
21
261
1581
